# Ranch Hand

AI operations assistant for Capitol Cowboys LLC — a D.C.-based young professional networking organization.

This is a live deployment for a real organization, not a demo project.

## Tech Stack

- **Backend:** Node.js + Express
- **AI:** Anthropic Claude API (claude-sonnet-4-20250514)
- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Hosting:** Railway

## Local Setup

```bash
git clone https://github.com/JohnJulian65/ranch-hand.git
cd ranch-hand
npm install
```

Create a `.env` file in the project root:

```
ANTHROPIC_API_KEY=your-api-key-here
```

Start the server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
